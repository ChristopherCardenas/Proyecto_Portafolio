Sobre mí

Mi nombre es Christopher Fabrizio Cárdenas Angulo, tengo 19 años y soy un apasionado desarrollador web. Me gusta crear páginas web interactivas y funcionales, utilizando tecnologías como HTML, CSS, JavaScript, Bootstrap, Tailwind CSS, Figma, Git, GitHub y bases de datos como SQL. Estoy comprometido con el aprendizaje continuo para ofrecer siempre buenos resultados en cada proyecto.

Proyecto zoológico Zoocax

Desarrollé el diseño de la página web informativa para el zoológico "Zoocax" utilizando Figma, y luego la implementé con HTML, CSS y Bootstrap 5. Gestioné el proyecto utilizando GitHub como plataforma de control de versiones, empleando Git para el seguimiento y control de cambios.

Proyecto Registro de alumnos

Además, gestioné el proyecto utilizando Git para el control de versiones y GitHub como plataforma de colaboración.
